Office module

In development mode (NODE_ENV=development), _classes node is displayed in the side menu. This item contains all application classes.

Go to the article menu. The main window displays a list of class objects. Create a new article.

Please note that a specific value is required in the status field (draft, published, archive), otherwise an error will occur when trying to save. It is not user-friendly. It is much better to let user choose from a predefined list. To do this, return to Studio, open the status attribute of the article class and set View type filed to Radio list. Export changes directly from the class form.

Office data is associated with metadata through element code names (class, attribute). If you delete attribute or change its code name, Office data will not be available, but will remain in storage. Metadata is an interface for Office data.
