Studio module

Studio creates metadata describing the structure of the application. Created data is in a separate storage. Metadata must be exported in order for it to load into the application.

You can export all data at once using Meta/Export top menu. Or you can export individual entities (classes, views, navigation) from editing form. This is faster, but metadata integrity may be compromised if changes were somewhere else.

Metadata will be saved in JSON format at specified path and automatically reloaded into serverâ€™s memory.

You can export anywhere into meta folder. However, the application will only be loaded from meta/app folder.
