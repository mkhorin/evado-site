Main page

Let's go to http://localhost:3000. The application boilerplate has several basic modules. Access to them is under hamburger menu icon and on the main page.

Application business logic structure (metadata - entities and relationships) is created in Studio module.

The structure is filled with data in Office module. This is basic version of the application operation.

App monitoring and managing is contained in Administration module.

Personal user data is displayed in Account module.

Plug-in modules are described in module section of config/default.json.

In addition to the above, there is API module. It is hidden from user navigation, but is available for direct AJAX requests to app.
