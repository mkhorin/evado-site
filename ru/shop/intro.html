Вступление

Создание приложения Интернет-магазин на декларативном фреймворке Evado.

Для начала посмотрите базовый урок создания приложения без написания кода.

Требуемая функциональность

  • Древовидная иерархия товарных категорий
  • Разные типы товаров с набором уникальных параметров
  • Выбор товара через параметры
  • Наполнение корзины для анонимных пользователей
  • Регистрация пользователя для оформления заказа
  • Автоматическое изменение остатков при создании или удалении заказа

Основные сущности приложения

  • Категория
  • Товар
  • Фото
  • Заказ

Исходный код готового приложения доступен в публичном репозитории.
