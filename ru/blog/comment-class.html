Класс Комментарий

Создайте класс comment. В нем будут храниться комментарии пользователей к статьям.

Создайте атрибут name (Имя) с типом Строка. Он будет хранить имя комментатора. Установите флажок Обязательный.

Создайте атрибут email с типом Строка. Он будет хранить электронный адрес комментатора. Установите флажок Обязательный. На форме атрибута перейдите на вкладку Валидация и добавьте валидатор Email. Он будет проверять строку на соответствие формату электронного адреса.

Создайте атрибут content (Содержимое) с типом Текст. Он будет хранить текст комментария. Установите флажок Обязательный. На форме атрибута перейдите на вкладку Валидация и добавьте валидатор Строка. Установите параметр Максимальная длина равным 256 символов.

Создайте атрибут article (Статья) с типом Ссылочный. Он будет хранить связь комментария со статьей. В поле Ссылочный класс выберите класс Статья. Установите флажок Обязательный.

Тип Обратная ссылка определяет связь через атрибут ссылочного класса. Если фотографии статьи извлекаются по ссылкам, которые хранятся в самой статье, то комментарии статьи извлекаются по ссылке на статью, которая хранится в комментарии.

Откройте класс Статья и создайте атрибут comments (Комментарии) с типом Обратная ссылка. В поле Ссылочный класс выберите класс Комментарий. В поле Ссылочный атрибут выберите атрибут Статья. Установите флажок Множественная связь. В статье может быть много комментариев. Уберите флажки команд Добавить и Убрать, потому что комментрий не может быть привязн к другой статье.

Экспортируйте метаданные и перейдите в Офис. Создайте статью и добавьте к ней комментарии.
