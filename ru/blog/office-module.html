Модуль Офис

В модуле Офис в режиме разработки NODE_ENV=development в боковом меню доступен пункт _classes, в котором отображаются все классы приложения.

Перейдите в класс Статья. В основном окне отобразится список объектов класса. Создайте новую статью.

Обратите внмание, что в поле Статус требуется ввести конкретное значение (draft, published, archived), иначе возникнет ошибка при сохранении объекта. Это неудобно. Гораздо лучше дать пользователю выбрать из предзаданного списка. Для этого вернитесь в Студию, откройте атрибут Статус класса Статья и измените тип представления на Радио-кнопки. Экспортируйте измененния прямо из формы класса.

Данные Офиса связаны с метаданными через кодовое имя элемента (класса, атрибута). Если вы удалите атрибут или измените имя, то данные в Офисе будут недоступны, но останутся в хранилище. Метаданные играют роль интерфейса для данных Офиса.
