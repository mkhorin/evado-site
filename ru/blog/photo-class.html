Класс Фото

Вернитесь в Студию и создайте класс photo. В нем будут храниться изображения к статьям.

Создайте атрибут file с типом Файл. Он будет хранить ссылку на файловое хранилище.

Создайте атрибут description (Описание) с типом Строка для описания файла.

Создайте атрибут orderNumber (Порядковый номер) с типом Целое. Он будет хранить номер для сортировки объектов в заданном порядке.

Класс Фото должен обладать специальным функционалом для загрузки и хранения файлов. Для этого выберите вкладку Поведения и добавьте поведение Файл. В настройках поведения установите флажок Только изображения.

Экспортируйте класс и перейдите в Офис. Создайте объекты фото, загружая файлы в приложение.
