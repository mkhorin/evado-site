Вступление

Создание приложения Электронное обучение на фреймворке Evado.

Для начала посмотрите базовый урок создания приложения без написания кода.

Требуемая функциональность

  • Разграничение прав доступа для учеников и учителей
  • Создание уроков, вопросов и заданий
  • Изменение текущего состояния задания
  • Комментирование заданий

Основные сущности приложения

  • Урок
  • Вопрос
  • Студент
  • Задание
  • Комментарий

Исходный код готового приложения доступен в публичном репозитории.
